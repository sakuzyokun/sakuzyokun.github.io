SakuZip (S-Zip)

SakuZip は 独自の .sz 形式を使った、手軽に使える圧縮＆解凍ツールです

GUI と CLI の両方を備え、Python だけで動作します (gzip + Base64 ベースのシンプル仕様)

主な機能

  • 独自拡張子 .sz に対応（中身は gzip + Base64）
  • GUI（Tkinter）で簡単に操作、進捗バーとログ表示あり
  • CLI でバッチ処理や自動化が可能
  • 軽量でコードが読みやすく、改造・学習用にも最適

使い方（GUI）

  1. Python をインストール（3.9 以上推奨）
  2. 必要なライブラリをインストール: pip install tqdm
  3. ダウンロードしたフォルダ内で python S-Zip_GUI.py を実行
  4. 「Extract / Compress」を選び、入力・出力を指定して Start を押す

使い方（CLI）

圧縮（フォルダ → .sz）:

python SZip_Compress.py <input_folder> <output_name>

解凍（.sz → フォルダ）:

python SZip_Extract.py <input.sz> <output_folder>

S-Zip フォーマット（簡易）

[Files]
{folder1/
file.txt
}
image.png

[folder1/file.txt]
<gzip_base64>

[image.png]
<gzip_base64>

※ テキストで中身が見えるのがこの形式の特徴です（Base64 により人間が開けられる）。

備考 / FAQ

  • 生成される .sz はテキスト形式のため、Base64 によるサイズ増加が起きます（小さめのファイルや学習用途向け）。
