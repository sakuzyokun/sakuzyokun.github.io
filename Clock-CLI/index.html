Clock-CLI

English Page

Clock は Windows Vista / 7 / 8 / 10 / 11 / Linux x64 / x86 / Arm64 / Mac OSX Arm64 / x64 で動作する、シンプルな時計です。

高速な動作を特徴としており、古いPCでも快適に実行できます。

主な機能

  • 今の時間 (1/1000 秒まで)
  • 今日の進捗
  • UTC・Unix時間

スクリーンショット

Windows 11 で Clock を起動した様子 (v1.0.0)
