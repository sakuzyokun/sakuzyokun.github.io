Clock-CLI

Japanese Page

Clock is a simple terminal-based clock application that runs on Windows Vista / 7 / 8 / 10 / 11, Linux x64 / x86 / Arm64, and macOS (Arm64 / x64).

It is designed for high performance and runs smoothly even on older PCs.

Main Features

  • Current time (up to 1/1000 second precision)
  • Today's progress indicator
  • UTC and Unix time
