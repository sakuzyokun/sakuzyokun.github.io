MessageBox

MessageBox は Windows XP / Vista / 7 / 8 / 10 / 11 で動作する、シンプルなアプリです。

高速な動作を特徴としており、古いPCでも快適に実行できます。

主な機能

  • エラーメッセージボックスの作成
  • ボタンの文字変更

スクリーンショット

Windows 11 で MessageBox を起動した様子 (v1.0.0)


注意

[ボタン]を[カスタム]にしている場合 ボタンが変な位置に生成される場合があります

その場合はメッセージ内容の最後にスペースを入れると正常に動作します

1. 変な位置だあ

2. スペース入れた後
