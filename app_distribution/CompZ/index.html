圧縮！！ZIPファイル！！

圧縮！！ZIPファイル！！ は Zipファイルと7zipファイルを圧縮/解凍できるソフトです

多くの機能と高速な動作を特徴としています

主な機能

  • zipと7zipの解凍/圧縮
  • 7zipのパスワードと圧縮率指定
  • 複数ファイル・フォルダの選択

使用方法

1. Pythonをインストール
(Win:MSStoreで「python」と検索し、お好きなものをインストール(絶対バージョン3以上) Mac:brew install python Linux: sudo apt install python)

2. 「pip install py7zr」と実行しpy7zrをインストールします

3. コンソール(コマンドプロンプト)でダウンロードフォルダに移動(cd Downloads で可能)し、「python3 "Compress!!_ZIPfile!!.py"」と打てば起動します

※もしほかにも「No module named 'ライブラリの名前'」と出た場合
1. pip install ライブラリの名前 とすると使えるようになります
