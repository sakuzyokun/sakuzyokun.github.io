System Monitor

日本語ページ

System Monitor is an app that gives you easy-to-understand information about your system.

It features high speed operation

Key Features

    == OS Info ==

  • Shows OS information and version
  • Shows CPU usage and memory information

    • == Process Info ==

  • Shows simple OS information, etc.
  • Shows a list of processes

How to use

1. Install Python
(Win: Search for "python" in MSStore and install your favorite (must be version 3 or higher) Mac: brew install python Linux: sudo apt install python)

2. In the console (command prompt), go to the download folder (you can do this with cd Downloads) and type "python3 file_info.py" to start it

3. If you want to start in GUI (window) mode, use "[the py file you want to start] --gui". If you want to change the update interval, use "[the py file you want to start] --interval [seconds]".

*If you get the message "No module named 'library name'"
1. pip install library name will allow you to use it
