YukkuriMovieMaker Installer

YukkuriMovieMaker Installer は ゆっくりMovieMakerを自動でダウンロードし、インストールまでしてくれるアプリです

使用方法

ダウンロードしたファイルを右クリック→管理者として実行 とすれば完了です
あとは自動でやってくれます

動作環境

Windowsのみで起動します (Wineでも起動しません)

Windows 8.1 以上で動作します

PowerShell 5.0以上が必要です(バージョン確認やインストール方法についてはこちらをご覧ください)

主な機能

  • ゆっくりMovieMaker4 最新版のインストール
  • デスクトップとスタートメニューにショートカットが作成できる
  • 完全自動でインストール開始

注意

1. 非公式です

2, 一定のバージョンからダウンロードする仕様ですので、インストール終了後ゆっくりMovieMakerを更新することをお勧めします

3. 下の画像のように「保護されました」と出る場合がありますが、詳細情報→実行　とすると実行できます
