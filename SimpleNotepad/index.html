SimpleStopwatch

SimpleNotepad は Windows Vista / 7 / 8 / 8.1 / 10 / 11 で動作する、軽量でシンプルなメモ帳です。

高速な動作を特徴としており、古いPCでも快適に実行できます。

名前募集中！！ Issuesに書き込め！！

主な機能

  • ファイルの開く / 保存
  • 全選択 / 切り取り / コピー / ペースト
  • Python版があるためMac・Linuxでも動作可能！

注意

1. Windows版の場合、下の画像のように「保護されました」と出る場合がありますが、詳細情報→実行　とすると実行できます


2. Python版の使い方

  1. 「Python版 ダウンロード」を押し、ファイルをダウンロードします
  2. コマンドプロンプト(または 端末、ターミナル など)を開き 「cd Downloads」と入力します
      3. できない場合は
    1. エクスプローラー(または Finder、ファイル など)でダウンロードフォルダ(Download、Downloads、ダウンロード など)を開く
    2. 右クリック(Macの場合は Ctrl、Control と書いてあるキーを押しながらクリック)をし、パスのコピー(または 上にあるパスが表示されているところを選択し、右クリック→コピー) (「端末で開く」がある場合は4まで飛ぶ)をする
    3. コマンドプロンプト(または 端末、ターミナル など)を開き、「cd "コピーしたパス"」と入力します
    4. これで完了です
  3. 「python SimpleNotepad.py」と入力しEnterを押すと実行できます！！
      4. pythonというコマンドが見つからないといわれた場合
    1. 「python3 SimpleNotepad.py」と入力してEnterを押す→これで実行されたらOK!!
    2. 無理だった場合はPythonがインストールされてない可能性があるので
      Mac / Windows の場合は公式サイトのダウンロードページにアクセスして、一番上の「Download the latest version for Windows」(もしくはMac)の部分の「Download Python [バージョン番号]」のボタンを押し、ダウンロードされたファイルをダブルクリックしてインストールを開始します
    3. Linuxの場合は ターミナル(もしくは 端末、コマンドプロンプト など)を開き「sudo apt install python」でPythonをインストールします
    4. これで完了です。終わったら「2. Python版の使い方」からやりましょう
